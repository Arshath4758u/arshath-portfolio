Patent

Provisional Specification Filed · 09 December 2025 · Indian Patents Act, 1970
JANUS: System and Method for Holistic, AI-Driven Thermal and Energy Optimization in Data Centers via Multi-Agent Reinforcement Learning
Co-inventors: Arshath Hameed K  ·  Akram M  ·  Aakash R
What is JANUS?

JANUS is a software-based predictive control platform for data center infrastructure. Unlike conventional systems that rely on reactive feedback loops (PID controllers or thermostats) — which only respond after a temperature spike has already occurred — JANUS uses a multi-agent AI architecture to predict thermal spikes before they happen and proactively adjust cooling.

The system models the entire data center as a directed graph where server racks are nodes, forecasts workload changes up to 48 hours in advance, and coordinates multiple AI agents to control cooling hardware, water valves, and job scheduling simultaneously.

7-Component Architecture
PiGNN Module
Physics-Informed Graph Neural Network for real-time airflow and thermal simulation — replaces expensive CFD simulations.
Transformer Module
Forecasts server workloads and weather impact up to 48 hours ahead, enabling proactive pre-cooling strategies.
MARL Module
Three cooperative agents (Thermal, Hydraulic, Workload) using CTDE — train centrally, execute independently in real time.
Causal AI Module
Performs root cause analysis and counterfactual "what-if" reasoning to learn from hypothetical failure scenarios.
XAI Interface
Translates AI decisions into human-readable audit logs for operator trust and regulatory compliance.
Hybrid Control Module
Dynamically switches between airside economization, Direct-to-Chip Liquid Cooling, and mechanical cooling.
Financial Module
SLO-aware job scheduling and battery arbitrage to minimize costs based on real-time electricity pricing.
